Игра «Крестики-нолики» с Telegram-ботом

##  Функциональность

### Геймплей
- Игрок играет **сердечками**, компьютер — **крестиками**.
- Простая и понятная механика — клик по пустой клетке делает ход.
- Компьютер отвечает через базовый алгоритм выбора свободной клетки.
- Определяется победа игрока, победа компьютера или ничья.

### Промокод при победе
- При победе игрока появляется **рандомный 5-символьный код**, состоящий из A–Z и 0–9.
- Промокод генерируется на клиенте при помощи функции `generatePromoCode()`.

### Telegram-интеграция
Backend отправляет результаты игры в Telegram-бота:

- При победе:
```

Победа! Промокод выдан: XXXXX

```
- При проигрыше:
```

Проигрыш

```

Отправка производится через PHP-скрипт `/api/send.php`, который вызывает Telegram Bot API методом POST.

## Как развернуть локально

### 1. Клонировать репозиторий
```bash
git clone https://github.com/duaves/tic-tac-toe.git
cd tic-tac-toe
````

### 2. Создать файл `.env` в корне проекта

```
BOT_TOKEN=ВАШ_ТОКЕН_БОТА
CHAT_ID=ВАШ_CHAT_ID
```

### 3. Запустить локальный PHP-сервер

```bash
php -S localhost:8000
```

### 4. Открыть игру

```
http://localhost:8000/
```

## Деплой

Приложение разворачивается через Dockerfile:

```dockerfile
FROM php:8-apache
COPY . /var/www/html/
EXPOSE 80
CMD ["apache2-foreground"]
```

Переменные окружения (Environment):

```
BOT_TOKEN = <токен бота>
CHAT_ID = <ID пользователя или чата>
```
